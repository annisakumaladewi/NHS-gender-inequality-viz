## Visualisations Included
The project includes four core visualisations:
- Line chart: Gender pay gap (mean vs median) over time  
- Line chart: Bonus pay gap (mean vs median)  
- Barbell chart: Bonus pay gap between women and men  
- Stacked bar chart: Ethnic composition of NHS staff by pay band  

All charts were designed with simplicity, readability, and editorial storytelling in mind.

## Data Sources
Data used in this project comes from publicly available sources, including:
- GOV.UK Gender Pay Gap Service (Barts Health NHS Trust)
- NHS Workforce Race Equality Standard (WRES)

The analysis focuses on aggregated trends rather than individual-level data.
